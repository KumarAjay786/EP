## URL Parameter Encoding

```
SPECIAL CHARACTERS IN QUERIES:

Space
  Plain Text:     Tamil Nadu
  URL Encoded:    Tamil%20Nadu    OR    Tamil+Nadu
  In Python:      urllib.parse.quote('Tamil Nadu')
  In JS:          encodeURIComponent('Tamil Nadu')

Ampersand
  Plain Text:     College & University
  URL Encoded:    College%20%26%20University
  Note:           Use %26 when it's part of value, not separator

Question Mark
  Plain Text:     What is this?
  URL Encoded:    What%20is%20this%3F
  Note:           %3F for literal ?, ? starts query string

Complete Example:
  Normal:    /api/colleges/list/?state=Tamil Nadu&search=IIT?
  Encoded:   /api/colleges/list/?state=Tamil+Nadu&search=IIT%3F
             OR
             /api/colleges/list/?state=Tamil%20Nadu&search=IIT%3F
```
